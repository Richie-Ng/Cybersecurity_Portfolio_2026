Cybersecurity Applied Activities Portfolio


Overview:
This repository serves as the evidence tracker and artifact collection for my applied cybersecurity portfolio. It contains scripts, code, screenshots, and documentation demonstrating the completion of 50 practical cybersecurity activities.


Repository Structure:
To maintain organization and align with submission requirements, the evidence is divided into two main directories:

* /Part\_1: Contains evidence for the first 25 foundational and investigative activities (e.g., physical security observations, cryptography implementation, and basic OSINT).
* /Part\_2: Contains evidence for the final 25 advanced activities (e.g., access control design, threat intelligence modules, and community cyber safety initiatives).


Evidence Tracking:
Progress is tracked continuously via GitHub commit history. Each piece of uploaded evidence (images, scripts, consent forms) is accompanied by a descriptive commit message linking it to its corresponding activity number (e.g., A1, B14). Final detailed explanations for each activity are compiled in the official PDF reports submitted separately.


Ethical Disclaimer:
All activities recorded in this repository were conducted ethically and strictly for educational purposes. Any testing involving vulnerabilities, prompt injection, or system bugs was performed exclusively on authorized, intentionally vulnerable educational targets or within strictly controlled local environments.
